# BlackJackGame
Creating a BlackJack Game using python, for 2 or more players. The game is a simple game where you have a deck of cards and eah player gets a card. The player could ask for more cards if needed. The player with the score closest to 21 would win the game

## Requirements
- Python 3.x

## Usage
To play the game just run the python script. Then you would choose the name of the players, and can modify the number of players in the main script. After chosing the name the script would ask you if you need more cards. To run the file:
- ./__main__.py
